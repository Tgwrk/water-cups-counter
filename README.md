# 💧 Water Tracker

A simple, responsive web app to track your daily water consumption.

## 📁 Files

- `index.html` – App layout and text elements.
- `index.css` – Centered layout, typography, and background styling.
- `index.js` – Core counter and saving logic.

## 🚀 How It Works

- **Increment**: Tap the button to add a cup to the current display.
- **Save**: Logs the current total into your history list and resets the counter to zero.
